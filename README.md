# Expense Analyzer (Python)

Небольшой учебный проект для портфолио: анализ расходов на основе списка операций (дата, категория, сумма).

## Что считает
- суммарные расходы по датам
- самый затратный день (дата и сумма)
- средний расход в день

## Как запустить
1) Откройте проект в PyCharm (или любой IDE)
2) Запустите файл `main.py`

## Пример входных данных
```python
expenditure_data = [
    {"date": "12.03.2025", "category": "milk", "amount": 130},
    {"date": "02.03.2025", "category": "fruits", "amount": 340},
]
```

## Пример вывода

{'12.03.2025': 850, '02.03.2025': 780, '12.09.2025': 300, '10.10.2025': 2200}  
10.10.2025: 2200  
1032.5
